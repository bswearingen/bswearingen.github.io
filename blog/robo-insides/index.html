In my last post I said that I wrote a “a simple Reddit bot to crawl through users’ post histories on demand, letting me know when and where they’re using far right shibboleths.” Technically speaking, that’s not 100% true. My robot is actually just a script running every ten minutes that parses a user’s 1000 most recent comments.

PRAW

The Python Reddit API Wrapper is, as the name says, a Python wrapper for the Reddit API. It makes developing a simple Reddit bot extremely easy. This block of code creates an instance of a Reddit object, checks all unread messages, and if the message is from me, runs a check on the user. 

import praw
reddit = praw.Reddit(client_id=os.environ[‘REDDIT_ID’],
                     client_secret=os.environ[‘REDDIT_SECRET’],
                     password=os.environ[‘REDDIT_PASS’],
                     user_agent=os.environ[‘REDDIT_AGENT’],
                     username=os.environ[‘REDDIT_USER’])
messages = reddit.inbox.unread(mark_read=True);
for rMessage in messages:
    rMessage.mark_read()
    if rMessage.author == “bswearin” and rMessage.subject != ‘comment reply’:
        try:
            if rMessage.subject == ‘username mention’:
                parent = reddit.comment(rMessage).parent()
                CheckUser(parent.author, rMessage)
            else:
                CheckUser(reddit.redditor(rMessage.body), rMessage)
        except:
            rMessage.reply(‘Error parsing: /u/’ + rMessage.body)

Checking Users

Most of the interesting stuff happens in the actual user check. To start, I break the giant list of slurs and shibboleths (not included in this post) into those that are single words and those that are multiple words. These are going to be checked for in different ways.

multi_word_slurs = [slur for slur in slurs if ' ' in slur]
single_word_slurs = [slur for slur in slurs if ' ' not in slur]

After establishing our shibboleths, we loop through all the available comments (Reddit limits it to 1000) and start processing. First, we remove quotes by deleting all lines that start with the ‘>’ character. Then we strip out any leading white space that might have been moved to the front of the post.

for comment in redditorSuspect.comments.new(limit=None):
    quoteFree = re.sub(r">.*\n", "", comment.body).strip('#\n \t')

After the comment is free from quotes, we start building a set of all shibboleths that it contains. A straight substring check is used to add in multi-word shibboleths, as they’re specific enough that an accidental match is unlikely.

Then we check the comment for echo parentheses.

Finally, we strip the punctuation, split the quote free comment into individual words, and check against our set of single word shibboleths.

contained = []
# Check against multi-word slurs, as they're unique enough to just do a string compare
contained.extend([slur for slur in multi_word_slurs if slur in quoteFree.lower()])
# Check against echo parentheses 
if('(((' in quoteFree and ')))' in quoteFree):
    contained.extend(['(((',')))'])
# Strip the punctuation, split the comment into single words, check against all our single word slurs
contained.extend([slur for slur in single_word_slurs if slur in quoteFree.translate(translator).split()])
